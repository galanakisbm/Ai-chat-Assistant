## 🔍 Προαπαιτούμενα

Πριν ξεκινήσετε την εγκατάσταση, βεβαιωθείτε ότι διαθέτετε:

### Τεχνικές Απαιτήσεις
- ✅ **PrestaShop**: Έκδοση 8.0.0 ή νεότερη
- ✅ **PHP**: Έκδοση 7.4 ή νεότερη
- ✅ **MySQL**: Έκδοση 5.6 ή νεότερη
- ✅ **SSL Certificate**: Συνιστάται (για ασφαλή σύνδεση με OpenAI)
- ✅ **Πρόσβαση FTP**: Για χειροκίνητη εγκατάσταση (προαιρετικά)

### Απαραίτητα για Λειτουργία
- 🔑 **OpenAI API Key**: Από το [platform.openai.com](https://platform.openai.com)
- 💳 **OpenAI Credit**: Πιστωτική ή χρεωστική κάρτα συνδεδεμένη στο OpenAI account

### Έλεγχος Έκδοσης PrestaShop

1. Συνδεθείτε στο **PrestaShop Admin Panel**
2. Πηγαίνετε στο **Advanced Parameters → Information**
3. Δείτε την έκδοση στο πεδίο **PrestaShop version**
4. Βεβαιωθείτε ότι είναι **8.0.0** ή μεγαλύτερη

## 🚀 Εγκατάσταση

### Μέθοδος 1: Εγκατάσταση μέσω Admin Panel (Συνιστάται)

#### Βήμα 1: Σύνδεση στο PrestaShop
1. Ανοίξτε το PrestaShop Admin Panel
2. Συνδεθείτε με τα διαπιστευτήριά σας

#### Βήμα 2: Μεταφόρτωση Module
1. Από το αριστερό μενού, πηγαίνετε στο:
   ```
   Modules → Module Manager
   ```
2. Πατήστε το κουμπί **"Upload a module"** (πάνω δεξιά)
3. Σύρετε το ZIP αρχείο ή κάντε κλικ για να το επιλέξετε
4. Περιμένετε μέχρι να ολοκληρωθεί το upload

#### Βήμα 3: Εγκατάσταση
1. Όταν ολοκληρωθεί το upload, θα δείτε το μήνυμα:
   ```
   Module uploaded successfully
   ```
2. Κάντε κλικ στο κουμπί **"Install"** δίπλα στο module
3. Περιμένετε την επιβεβαίωση εγκατάστασης

#### Βήμα 4: Ενεργοποίηση
1. Αφού ολοκληρωθεί η εγκατάσταση, κάντε κλικ στο **"Configure"**
2. Προχωρήστε στο [Διαμόρφωση](#διαμόρφωση)

---

### Μέθοδος 2: Χειροκίνητη Εγκατάσταση μέσω FTP

#### Βήμα 1: Σύνδεση μέσω FTP

**Με FileZilla:**
1. Κατεβάστε και εγκαταστήστε το [FileZilla Client](https://filezilla-project.org/)
2. Ανοίξτε το FileZilla
3. Συμπληρώστε τα στοιχεία σύνδεσης:
   - **Host**: ftp.yourdomain.com (ή η IP διεύθυνση)
   - **Username**: Το FTP username σας
   - **Password**: Το FTP password σας
   - **Port**: 21 (ή 22 για SFTP)
4. Πατήστε **"Quickconnect"**

#### Βήμα 2: Μεταφορά Αρχείων
1. Στο **τοπικό panel** (αριστερά), πλοηγηθείτε στο φάκελο που κατεβάσατε το module
2. Στο **απομακρυσμένο panel** (δεξιά), πλοηγηθείτε στο:
   ```
   /public_html/modules/
   ```
   ή
   ```
   /www/modules/
   ```
3. Αποσυμπιέστε το ZIP αρχείο τοπικά
4. Σύρετε ολόκληρο το φάκελο `optic_aichat` στο `/modules/`
5. Περιμένετε μέχρι να ανεβούν όλα τα αρχεία

#### Βήμα 3: Ρύθμιση Δικαιωμάτων (Permissions)
1. Κάντε δεξί κλικ στο φάκελο `optic_aichat`
2. Επιλέξτε **"File permissions"** ή **"CHMOD"**
3. Ορίστε δικαιώματα σε **755** ή **775**
4. Επιλέξτε **"Recurse into subdirectories"**
5. Κάντε κλικ **"OK"**

#### Βήμα 4: Εγκατάσταση από Admin
1. Πηγαίνετε στο PrestaShop Admin → **Modules → Module Manager**
2. Αναζητήστε "OpticWeb AI Chat" ή "optic_aichat"
3. Κάντε κλικ στο **"Install"**
4. Κάντε κλικ στο **"Configure"**

## ⚙️ Διαμόρφωση

### Βήμα 1: Λήψη OpenAI API Key

#### 1.1 Δημιουργία Λογαριασμού OpenAI
1. Πηγαίνετε στο [platform.openai.com](https://platform.openai.com)
2. Κάντε κλικ στο **"Sign Up"** (αν δεν έχετε λογαριασμό)
3. Συμπληρώστε email, κωδικό και επιβεβαιώστε το email σας

#### 1.2 Προσθήκη Πληρωμής
1. Πηγαίνετε στο **Settings → Billing**
2. Κάντε κλικ στο **"Add payment method"**
3. Προσθέστε πιστωτική/χρεωστική κάρτα
4. Προσθέστε τουλάχιστον **$5-10 credit** για αρχή

#### 1.3 Δημιουργία API Key
1. Από το μενού, πηγαίνετε στο **API keys**
2. Κάντε κλικ στο **"Create new secret key"**
3. Δώστε ένα όνομα (π.χ. "PrestaShop Chat")
4. **ΠΡΟΣΟΧΗ**: Αντιγράψτε το key ΑΜΕΣΩΣ - δεν θα το ξαναδείτε!
5. Το key αρχίζει με `sk-...`

### Βήμα 2: Ρύθμιση Module

#### 2.1 Πρόσβαση στις Ρυθμίσεις
1. PrestaShop Admin → **Modules → Module Manager**
2. Αναζητήστε "OpticWeb AI Chat"
3. Κάντε κλικ στο **"Configure"**

#### 2.2 Συμπλήρωση Πεδίων

**1. Chat Widget Title** (Τίτλος Widget)
- Εισάγετε τον τίτλο που θα εμφανίζεται στο chat
- Παράδειγμα: "Βοηθός Καταστήματος" ή "OpticWeb Assistant"
- Αυτό θα φαίνεται στην κορυφή του chat window

**2. OpenAI API Key**
- Επικολλήστε το API key που αντιγράψατε
- Μορφή: `sk-proj-xxxxxxxxxxxxxxxxxxxxx`
- Βεβαιωθείτε ότι δεν υπάρχουν κενά πριν/μετά

**3. System Prompt (Οδηγίες AI)**
- Δώστε οδηγίες στο AI για το πώς να συμπεριφέρεται
- Παράδειγμα:
  ```
  Είσαι ο ψηφιακός βοηθός του καταστήματος "OpticWeb".
  Είμαστε ειδικευμένοι σε γυαλιά ηλίου και οπτικά.
  Έχουμε δωρεάν μεταφορικά άνω των 50€.
  Οι παραγγελίες παραδίδονται σε 1-3 εργάσιμες.
  Απάντα πάντα στα Ελληνικά, φιλικά και σύντομα.
  ```

**4. Enable Page Context** (Ενεργοποίηση Πληροφοριών Σελίδας)
- Ενεργοποιήστε: **Yes**
- Αυτό επιτρέπει στο AI να βλέπει σε ποια σελίδα βρίσκεται ο χρήστης
- Παράδειγμα: Αν ο χρήστης είναι σε σελίδα προϊόντος, το AI θα το γνωρίζει

**5. Page Context Template** (Προαιρετικό)
- Αφήστε κενό για default λειτουργία
- Ή δημιουργήστε custom template με μεταβλητές:
  ```
  Σελίδα: {PAGE_TITLE}
  URL: {PAGE_URL}
  Τύπος: {PAGE_TYPE}
  Προϊόν: {PRODUCT_NAME}
  Τιμή: {PRODUCT_PRICE}
  ```

#### 2.3 Αποθήκευση
1. Κάντε κλικ στο κουμπί **"Αποθήκευση"** (Save)
2. Θα δείτε μήνυμα επιτυχίας: "Οι ρυθμίσεις αποθηκεύτηκαν"

## ✅ Έλεγχος Λειτουργίας

### Τεστ 1: Εμφάνιση Widget
1. Πηγαίνετε στην κεντρική σελίδα του e-shop σας
2. Πατήστε **Ctrl+Shift+R** (hard refresh) για να φορτώσει το νέο module
3. Θα πρέπει να δείτε το **chat button** (💬) κάτω δεξιά
4. Κάντε κλικ για να ανοίξει το chat

### Τεστ 2: Απλή Ερώτηση
1. Γράψτε: "Γεια σου"
2. Πατήστε Enter
3. Θα πρέπει να δείτε:
   - **"Thinking..."** για λίγα δευτερόλεπτα
   - Μετά την απάντηση του AI στα Ελληνικά

### Τεστ 3: Αναζήτηση Προϊόντων
1. Γράψτε: "Δείξε μου γυαλιά ηλίου"
2. Το AI θα πρέπει να:
   - Αναζητήσει προϊόντα
   - Εμφανίσει λίστα με εικόνες και τιμές

### Τεστ 4: Quick Reply Buttons
1. Κάντε κλικ σε ένα από τα γρήγορα κουμπιά (π.χ. "🏷️ Προσφορές")
2. Θα πρέπει να στείλει αυτόματα το μήνυμα

### Τεστ 5: Mobile
1. Ανοίξτε το site από κινητό ή tablet
2. Το chat θα πρέπει να καλύπτει ολόκληρη την οθόνη
3. Δοκιμάστε να στείλετε μήνυμα

## 🐛 Συνήθη Προβλήματα

### Πρόβλημα 1: Το chat δεν εμφανίζεται

**Αιτία**: Το module δεν είναι ενεργοποιημένο ή υπάρχει πρόβλημα με το cache

**Λύση**:
1. **Modules → Module Manager** → Αναζήτηση "OpticWeb"
2. Βεβαιωθείτε ότι δείχνει **"Installed"**
3. Πηγαίνετε στο **Advanced Parameters → Performance**
4. Κάντε κλικ στο **"Clear cache"**
5. Επιλέξτε **"Clear cache"** και **"Yes"**
6. Κάντε hard refresh (Ctrl+Shift+R) στο front-end

---

### Πρόβλημα 2: "Configuration Error: API Key missing"

**Αιτία**: Το API Key δεν έχει αποθηκευτεί σωστά

**Λύση**:
1. Πηγαίνετε στο **Modules → Configure**
2. Ελέγξτε ότι το **OpenAI API Key** πεδίο είναι συμπληρωμένο
3. Το key πρέπει να ξεκινά με `sk-`
4. Αποθηκεύστε ξανά
5. Καθαρίστε το cache

---

### Πρόβλημα 3: "OpenAI Error: Insufficient credits"

**Αιτία**: Δεν έχετε πιστώσεις στο OpenAI account

**Λύση**:
1. Πηγαίνετε στο [platform.openai.com/account/billing](https://platform.openai.com/account/billing)
2. Ελέγξτε το **Account balance**
3. Αν είναι $0, πατήστε **"Add to credit balance"**
4. Προσθέστε τουλάχιστον $5-10

---

### Πρόβλημα 4: "OpenAI Error: Rate limit exceeded"

**Αιτία**: Πολλά requests σε σύντομο χρονικό διάστημα

**Λύση**:
1. Περιμένετε 1-2 λεπτά
2. Δοκιμάστε ξανά
3. Αν συνεχιστεί, ελέγξτε το tier του λογαριασμού σας στο OpenAI

---

### Πρόβλημα 5: Αργές απαντήσεις (>10 δευτερόλεπτα)

**Αιτία**: Φόρτος στο OpenAI ή αργό internet

**Λύση**:
1. Ελέγξτε την ταχύτητα internet (speedtest.net)
2. Μειώστε το μέγεθος του **System Prompt** (κρατήστε το <200 λέξεις)
3. Ελέγξτε το status του OpenAI: [status.openai.com](https://status.openai.com)

---

### Πρόβλημα 6: Το AI δεν απαντά στα Ελληνικά

**Αιτία**: Το System Prompt δεν έχει σαφείς οδηγίες

**Λύση**:
1. Πηγαίνετε στο **Configure → System Prompt**
2. Προσθέστε στην αρχή:
   ```
   ΣΗΜΑΝΤΙΚΟ: Απάντα ΠΑΝΤΑ στα Ελληνικά (Greek language).
   ```
3. Αποθηκεύστε

---

### Πρόβλημα 7: Database error κατά την εγκατάσταση

**Αιτία**: Πρόβλημα με δικαιώματα MySQL ή υπάρχων πίνακας

**Λύση**:
1. Απεγκαταστήστε το module
2. Συνδεθείτε στο **phpMyAdmin**
3. Διαγράψτε τον πίνακα (αν υπάρχει):
   ```sql
   DROP TABLE IF EXISTS ps_optic_aichat_logs;
   ```
4. Εγκαταστήστε ξανά το module
